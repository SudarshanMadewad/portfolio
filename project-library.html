LIBRARY MANAGEMENT

Dashboard Design

PROJECT

Library Management Dashboard

ROLE

UI/UX Designer

TIMELINE

5 Weeks

TOOLS

Figma, FigJam, Maze

SKILLS
Dashboard Design Data Visualization UX Research Design Systems
01 — PROBLEM

THE CHALLENGE

University librarians struggled with outdated management tools that required manual data entry, offered no analytics, and had poor usability. The existing system was DOS-based and created significant workflow inefficiencies.

02 — RESEARCH

UNDERSTANDING USERS

Shadowed 4 librarians across 2 institutions, documenting their daily workflows. Identified key tasks: book cataloging, student borrowing management, overdue tracking, and inventory reporting.

03 — WIREFRAMES

INFORMATION ARCHITECTURE

Designed the dashboard around the most frequent tasks: quick book search, borrowing status overview, and expiry alerts. Created modular widget-based layouts for customizable dashboards.

WIREFRAME ITERATIONS Dashboard Layouts
04 — UI DESIGN

VISUAL SYSTEM

Built a clean, data-dense design system optimized for daily use. Color coding for status indicators, clear data hierarchy, and scannable typography ensure librarians can process information quickly.
