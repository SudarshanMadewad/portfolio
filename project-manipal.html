MANIPAL UNIVERSITY

Landing Page Redesign

PROJECT

Manipal University Landing Page

ROLE

UI/UX Designer

TIMELINE

4 Weeks

TOOLS

Figma, FigJam, Adobe Illustrator

SKILLS
UX Research Wireframing UI Design Prototyping Design Systems
01 — PROBLEM

THE CHALLENGE

Manipal University's existing landing page failed to effectively communicate the institution's value proposition to prospective students. The outdated design lacked visual hierarchy, had poor information architecture, and didn't reflect the university's modern, research-driven identity.

Key issues included low conversion rates on application forms, high bounce rates from the homepage, and negative feedback from user surveys about navigation and content discoverability.

02 — RESEARCH

DISCOVERY & INSIGHTS

Conducted user interviews with 15 prospective students and 5 faculty members. Performed competitive analysis of 8 leading university websites globally. Created user journey maps and identified key pain points in the enrollment funnel.

03 — WIREFRAMES

STRUCTURE & LAYOUT

Developed low-fidelity wireframes exploring multiple layout approaches. Prioritized content hierarchy based on user research findings, placing program discovery and application CTAs at strategic touchpoints.

LOW-FIDELITY WIREFRAMES Desktop & Mobile Layouts
04 — UI DESIGN

VISUAL SYSTEM

Created a comprehensive design system with a refreshed color palette, typography scale, and component library. The visual direction balances academic authority with modern approachability.

DESIGN SYSTEM Colors, Typography & Components
05 — FINAL SCREENS

THE SOLUTION

The final design features a dynamic hero section with program highlights, streamlined navigation, interactive campus tour elements, and a simplified application flow that reduced steps from 7 to 3.

06 — RESULTS

IMPACT

The redesigned landing page saw significant improvements across all key metrics during A/B testing against the original design.
